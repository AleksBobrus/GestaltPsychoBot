# 🤖 GestaltPsychoBot – Telegram-бот психологической поддержки

[![Telegram](https://img.shields.io/badge/Telegram-@GestaltPsychoBot-blue?style=flat&logo=telegram)](https://t.me/GestaltPsychoBot)  

Телеграм-бот, сочетающий принципы гештальт-терапии и современные технологии искусственного интеллекта. Проводит диагностику (тест Бека), предлагает терапевтические упражнения, обеспечивает кризисную поддержку и ведёт осознанный диалог с помощью OpenAI.

---

## 📋 Функционал

- **Психологическая диагностика:** прохождение теста Бека на уровень депрессии с подробной интерпретацией результатов.
- **Терапевтические упражнения:** письменные практики из гештальт-терапии («Пустой стул», «Письмо благодарности», «Работа с внутренним критиком» и другие).
- **Кризисная поддержка:** быстрый доступ к контактам экстренной психологической помощи и техники самопомощи при панических атаках.
- **Диалог с ИИ-психологом:** общение с моделью OpenAI, настроенной на эмпатичный, поддерживающий стиль с элементами гештальт-подхода.
- **Интерактивное меню:** удобная навигация с помощью кнопок (aiogram 3.x).
- **Хранение данных:** прогресс и ответы пользователя сохраняются в базе данных SQLite.

---

## 🛠️ Технический стек

- **Язык:** Python 3.10+
- **Фреймворк:** Aiogram 3.x (асинхронный)
- **База данных:** SQLite + aiosqlite (асинхронный доступ)
- **Искусственный интеллект:** OpenAI API (GPT-3.5-turbo / GPT-4)
- **Конфигурация:** python-dotenv, Pydantic Settings
- **Логирование:** стандартный модуль logging

## 📁 Структура проекта

```txt
GestaltPsychoBot/
├── .env.example              # образец конфигурационного файла
├── requirements.txt          # список зависимостей
├── main.py                   # точка входа, запуск поллинга бота
├── config.py                 # настройки бота и OpenAI (Pydantic)
├── handlers/                 # обработчики команд и сообщений
│   ├── start.py              # /start, приветствие
│   ├── menu.py               # главное меню
│   ├── beck_test.py          # логика теста Бека
│   ├── exercises.py          # упражнения и письменные практики
│   ├── crisis.py             # кризисная поддержка
│   └── ai_dialog.py          # общение с ChatGPT
├── keyboards/                # клавиатуры и инлайн-кнопки
├── database/                 # модули работы с SQLite
│   ├── models.py             # описание таблиц
│   └── repository.py         # CRUD-запросы
├── utils/                    # вспомогательные функции
├── tests/                    # (опционально) юнит-тесты
└── screenshots/              # скриншоты для README
```
